🔐 Güvenilir Şifrem - Modern Password Generator
Bu proje, Python'un customtkinter kütüphanesi kullanılarak geliştirilmiş, modern görünümlü ve kullanıcı dostu bir şifre oluşturma uygulamasıdır. Kullanıcıların belirledikleri uzunlukta karmaşık şifreler üretmesini ve bu şifreleri kolayca yönetmesini sağlar.

✨ Özellikler
Modern Arayüz: customtkinter ile pembe/mavi tonlarında estetik bir tasarım.

Dinamik Uzunluk: Kaydırma çubuğu (slider) ile 6 ile 32 karakter arasında şifre belirleme.

Tek Tıkla Kopyala: Üretilen şifreyi anında panoya kopyalama özelliği.

Masaüstüne Kayıt: Şifreleri hangi platform için oluşturduğunuzu belirterek masaüstündeki sifrelerim.txt dosyasına otomatik kaydetme.

Güvenli Karakterler: Harf, rakam ve özel karakterlerden oluşan güçlü kombinasyonlar.

🛠️ Kullanılan Teknolojiler
Dil: Python

Arayüz: customtkinter (Modern Tkinter Wrapper)

Kütüphaneler: os, random, string, tkinter

🚀 Kurulum ve Çalıştırma
Uygulamayı bilgisayarınızda çalıştırmak için şu adımları izleyebilirsiniz:

Gerekli kütüphaneyi yükleyin:

pip install customtkinter

Projeyi klonlayın veya indirin.

Çalıştırın:

python python.py
