# G-venliSifrem
Güvenilir Şifrem: Python ve CustomTkinter kütüphanesi kullanılarak geliştirilmiş, modern ve şık arayüzlü bir güvenli şifre oluşturma uygulamasıdır. Kullanıcı dostu slider ile şifre uzunluğu belirlenebilir, tek tıkla kopyalama yapılabilir ve oluşturulan şifreler "sifrelerim.txt" olarak masaüstüne kaydedilebilir.
